# Gibbiverse

## Summary

This repository contains the Static Website Generator for the Gibbiverse website. It contains the main Hugo configuration. The blog content itself can be found in the following [repsoitory](https://github.com/ScottGibb/blog-posts).

## Pull Requests
If you wish to make pull requests or make changes to my blogs, you are more than welcome to submit a PR in the blog content [repsoitory](https://github.com/ScottGibb/blog-posts). All contributions are welcome.
